## 💻 Funcionalidades

* Como colocar o projeto em localhost ?
  no terminal do projeto de o seguinte comando node
```
  node server
```

o seguinte comando ira colocar o projeto na localhost 8080

</br>

* como observar ?
  Coloque na barra do seu browser
```
  http://localhost:8080/
```
